# PFAD Voronoi Exploration
Goldsmiths MA Computational Arts Term 1 Module. <br />
Programming for Artists and Designers mid-term asssessment python project

## Installation
### Versions
Python 3.12.3 <br />
py5

### Libraries

Installing with Conda

[numpy](https://numpy.org/install/)
```bash
conda install numpy
```
[noise](https://pypi.org/project/noise/)
```bash
conda install noise
```
[py5canvas](https://pypi.org/project/py5canvas/)
```bash
conda install py5canvas
```
[scipy](https://scipy.org/install/)
```bash
conda install scipy
```
More information about this scipy.spatial library can be found [here](https://docs.scipy.org/doc/scipy/reference/spatial.html#module-scipy.spatial)

## Usage

Run on termimal
```bash
python mid-term-assessment-final.py
```
Or

Run using play button in Visual Studio Code
